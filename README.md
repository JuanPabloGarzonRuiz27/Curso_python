# Sistema de Vehículos en Python - Programación Orientada a Objetos

## Descripción del sistema

Este proyecto consiste en el desarrollo de un sistema sencillo utilizando **programación orientada a objetos (POO)** en Python. El programa modela diferentes tipos de vehículos mediante el uso de **clases, objetos y herencia**.

El sistema permite representar vehículos como **carros y motos**, mostrando sus características principales como marca, modelo, línea y otras propiedades específicas de cada tipo de vehículo.

El objetivo del proyecto es demostrar el uso de los conceptos  de la programación orientada a objetos para organizar y estructurar el código de forma clara y reutilizable.

### Clases del sistema

El sistema contiene tres clases principales:

**Vehiculo**

* Clase base del sistema.
* Contiene los atributos generales de cualquier vehículo.

**Carro**

* Hereda de la clase Vehiculo.
* Representa automóviles e incluye el número de puertas.

**Moto**

* Hereda de la clase Vehiculo.
* Representa motocicletas e incluye el cilindraje.

## Cómo ejecutar el programa

Para ejecutar el programa se deben seguir los siguientes pasos:

1. Instalar Python en el computador.
2. Descargar o clonar el repositorio.
3. Abrir una terminal o consola.
4. Ejecutar el siguiente comando:

```
python exercise_basico_POO.py
```

Al ejecutarlo, el programa mostrará en la consola la información de los vehículos registrados en el sistema.
